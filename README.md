安装方法：

    -解压下载的全部压缩包，复制Glyde-ZH_p.pak
    -定向到文件夹"X:\Steam\steamapps\common\Glyde The Dragon™ Demo\Glyde\Content\Paks"
    -在该目录的空白位置粘贴。
    -完成。
